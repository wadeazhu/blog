5、安装包没有问题，但是运行出现了问题

这个是由于import 导入组件出了问题，需要改成下面的写法

component: () => import("@/views/${this.compName}.vue") // 不能这么写

component: resove => require([`@/views/${this.compName}.vue`], resolve)  // 要这么写
